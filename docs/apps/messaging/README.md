## What this app is (user-facing)

A messaging app for direct and group conversations, with file sharing and optional voice notes.

## Success criteria

- Web-first UX
- Simple self-hosting and admin
- Fast message sync and search
- File attachments, including audio files

## Data ownership expectations

- Export of conversations and attachments
- Clear data retention and deletion controls
